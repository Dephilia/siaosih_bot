# 笑死(siaosih)

Plurk reply bot.
siaosih, haha

## Usage

Go ask plurk a bot token, an account.

`cp token.txt.example token.txt` and make sure your `token.txt` has well modified.

Install **docker**.

```
docker-compose up -d --build
```

If no, install **pipenv**.

```
pipenv run python main.py
```


Nothing more.
